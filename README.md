# vue-image-picker

![Vue Image Picker Demo](http://i.giphy.com/l0HlVYp5QVO9dYcgw.gif)

## Usage

```html
<template>
  <div>
    <image-picker></image-picker>
  </div>
</template>

<script>
  import ImagePicker from 'vue-image-picker'

  export default {
    components: {
      'image-picker': ImagePicker
    }
  }
</script>
```
